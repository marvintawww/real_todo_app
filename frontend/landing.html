Возможности

Всё нужное — ничего лишнего

Taska построена вокруг одной идеи: минимум трений между вами и вашими задачами.

Недельный календарь

Горизонтальный пикер показывает 7 дней сразу. Переключайтесь по неделям одним кликом, фильтруйте задачи по конкретному дню — всё моментально.

Умные задачи

У каждой задачи есть название, описание, дата и тип. Отмечайте выполненные прямо из карточки — статус обновляется без перезагрузки страницы.

Быстрый поиск и фильтры

Поиск с дебаунсингом находит задачи на лету. Фильтруйте по статусу выполнения, дате или типу — комбинируйте условия как угодно.

Типы задач

Создавайте собственные категории: «Работа», «Личное», «Учёба» — любые. Каждая задача привязана к типу, что помогает видеть картину целиком.

Светлая и тёмная тема

Переключение темы одним кликом. Выбор сохраняется между сессиями — приложение запоминает ваши предпочтения локально.

Аккаунты и авторизация

Регистрация и вход по логину и паролю. Все задачи привязаны к вашему аккаунту — данные надёжно хранятся на сервере через JWT-авторизацию.

Как работает

Четыре шага до первой задачи

Никакой настройки, никаких туториалов. Просто зайдите и начните.

01
Создайте аккаунт
Придумайте логин и пароль. Никакой почты, никаких подтверждений — один экран и вы внутри.
02
Добавьте типы
Перейдите в раздел «Типы задач» и создайте нужные категории: работа, учёба, спорт — всё что угодно.
03
Создайте задачу
Нажмите «Новая задача», заполните название, выберите тип и укажите дату. Описание — по желанию.
04
Выполняйте
Отмечайте задачи выполненными прямо из карточки. Фильтруйте по дням через календарь сверху.
Легенда

Что означает каждый элемент

Разбираем интерфейс по деталям — чтобы было понятно с первой секунды.

Недельный пикер дат

Полоса из 7 кнопок-дней в верхней части раздела задач. Показывает текущую неделю с понедельника по воскресенье.

  • Число в кружке — день месяца. Тонкая обводка кружка = сегодня.
  • Закрашенный кружок = выбранный день, задачи фильтруются по нему.
  • Стрелки ‹ › справа переключают неделю назад или вперёд.
  • Кнопка «Сегодня» появляется только когда вы ушли от текущей недели.
  • Повторный клик по выбранному дню снимает фильтр и показывает все задачи.

Карточка задачи

Основной элемент приложения. Каждая карточка — одна задача с набором атрибутов.

  • Чекбокс в правом углу — нажмите, чтобы отметить задачу выполненной.
  • Синий бейдж внизу — тип задачи, назначенный при создании.
  • Дата справа — день, на который запланирована задача.
  • Выполненные задачи уменьшают прозрачность и зачёркивают название.
  • Кнопки «Редактировать» и «Удалить» — управление прямо из карточки.

Типы задач

Пользовательские категории для классификации задач. Доступны в разделе «Типы задач» через навигацию.

  • Тип обязателен при создании задачи — без него сохранение недоступно.
  • Введите название и нажмите «Добавить» — тип появится в списке мгновенно.
  • Каждый тип отображается как бейдж в карточке задачи.
  • Удаление типа — крестик рядом с его названием в разделе управления.

Фильтры и поиск

Панель под пикером дат. Помогает быстро найти нужную задачу среди всех.

  • Текстовое поле — поиск по названию задачи, срабатывает с задержкой 300 мс.
  • Выпадающий список — фильтр по статусу: все, выполненные, невыполненные.
  • Все фильтры работают одновременно и комбинируются между собой.
  • Кнопка «Сбросить» снимает сразу все фильтры включая выбранный день.
